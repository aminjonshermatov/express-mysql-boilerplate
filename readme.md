## Express MySQL boilerplate
Simple boilerplate with **jwt**, hash password using **bcrypt** and **auth middleware**

***Endpoints:***

 - ***./api/auth/register***
- ***./api/auth/login***

Before start don't forget to create **.env** file in root folder. Requirements for **.env** file:

 - **PORT**
 - **NODE_ENV**
 - **SECRET_KEY**
 - **HOST**
 - **USER**
 - **PASSWORD**
 - **DB**
